# 🌶️ Chuleta — іспанська шпаргалка

> **Chuleta** (чулета) — іспанський сленг для «шпаргалки». Саме те, що це є

Веб-аплікуха для вивчення іспанської граматики і словника. Стиль — паперовий блокнот з кільцевим переплетом, все компактно і по суті

🔗 **[Відкрити аплікуху](https://sh-kirka.github.io/chuleta-app/)**

---

## Що всередині

### ⏰ Часи (Tiempos)
Усі способи і часи іспанської — з поясненням **коли використовувати**, маркерами часу і таблицями відмінювання. Вкладений акордеон з нерегулярними формами прямо всередині кожного часу.

| Modo | Tiempos |
|---|---|
| **Indicativo** | Presente · Perfecto · Indefinido · Imperfecto · Pluscuamperfecto · Futuro · Futuro Perfecto |
| **Condicional** | Simple · Compuesto |
| **Subjuntivo** | Presente · Imperfecto · Perfecto · Pluscuamperfecto |
| **Imperativo** | Afirmativo · Negativo |

### ⚡ Дієслова (Verbos)
- Verbos reflexivos (levantarse, ducharse...)
- Perífrasis verbales (ir a, tener que, acabar de, seguir, dejar de...)
- Gerundio — форми і вживання
- Se pasiva / Se impersonal
- Нерегулярні форми yo в Presente

### 🔤 Граматика
- SER vs ESTAR — коли що
- GUSTAR і подібні (encantar, doler, faltar...)
- Artículos (el/la + виключення el agua)
- Interrogativos (qué vs cuál — вічна плутанина!)
- Negación подвійна (nada, nadie, nunca, tampoco...)
- Comparación (más/menos que, tan como, mejor/peor)
- Прийменники + POR vs PARA + транспорт (a pie, en coche...)
- Conectores del discurso
- Objeto Directo / Indirecto (me/te/lo/la/le + порядок)
- Voz Pasiva vs Activa
- Oraciones Condicionales (3 типи Si...)
- Pronombres Relativos (que, quien, donde, cuyo...)
- Estilo Indirecto — зміна часів і маркерів

### 📖 Словник (Vocabulario)

`Їжа` `Напої` `Транспорт` `Емоції` `Характер` `Дім` `Одяг` `Сім'я` `Робота` `Місто` `Природа` `Тіло` `Числа` `Дні/Місяці` `Здоров'я` `Погода` `Кольори` `Подорожі` `Спорт/Хобі` `Дієслова` `Прикметники` `Ресторан`

## Фічі

- 🔍 **Пошук** по всьому — граматика + словник одночасно, підсвічування знайденого
- 🇺🇦 🇪🇸 **Три мови** — УКР / ENG / ESP перемикаються в хедері
- 📂 **Акордеон** — розкриваєш тільки те що потрібно
- ⚡ **Вкладені нерегулярні** — прямо всередині картки часу
- 📱 **PWA** — можна встановити на телефон як аплікуху (iOS: Safari → Поділитись → На головний екран)

## Технічно

Чистий HTML/CSS/JS — без фреймворків, без залежностей. Один файл `index.html`.

Шрифти: [Caveat](https://fonts.google.com/specimen/Caveat) (рукописний) + [DM Sans](https://fonts.google.com/specimen/DM+Sans).
